start(self)

source code 

Start running the Cocos GUI. This method will block until the GUI has been shut down. After returning there might still be some processes cleaning up.

show_ticker_scene(self, new_ticker=False)

source code 

Replace everything on screen with the scene displaying the drink ticker. If the engine is not yet running, this method will start a new engine and will block until the engine has stopped.

_display_scene(self, scene)

source code 

Change the display to show the given scene. If no engine is running yet, an engine will be started. In that case this method will not return until the engine has stopped again.
