draw_gradient(draw, rail1_start, rail1_end, rail2_start, rail2_end, start_color, end_color, gradient_start=0.0, gradient_end=1.0)

source code 

Draw a gradient sweeping over two rails. draw = ImageDraw object rail1_start = Start of first rail (x,y) rail1_end = End of first rail (x,y) rail2_start = Start of second rail (x,y) rail2_end = End of second rail (x,y) start_color = color at the start of the gradient end_color = color at the end of the gradient. gradient_start = factor where to start the gradient (0.0 is start, 1.0 = end) gradient_end = factor where to end the gradient (0.0 is start, 1.0 = end)
