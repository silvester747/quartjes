classtools Module¶

class quartjes.util.classtools.AttrDisplay[source]¶

Bases: object

Provides an inheritable print overload method that displays instances with their class names and a name=value pair for each attribute stored on the instance itself (but not attrs inherited from its classes). Can be mixed into any class, and will work on any instance.

class quartjes.util.classtools.QuartjesBaseClass(id=None)[source]¶

Bases: quartjes.util.classtools.AttrDisplay

Base class for all objects in quartjes that are serialized. Deriving from this class does not automatically make your objects serializable, but at least a unique id is present.
